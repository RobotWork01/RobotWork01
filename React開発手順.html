■開発環境の構築（インストール）
 1.テキストエディタ（推奨：Visual Studio Code）
 2.Node.js： JavaScriptをブラウザ以外でも実行できるようにするソフトウェアです。
   Reactの開発には必須です。
 3.npm（Node Package Manager）： Node.jsに付属する、ライブラリ管理ツールです。
   ReactやほかのJavaScriptライブラリをインストールするのに使います。


■開発手順 参考URL：https://www.choge-blog.com/programming/windows-install-react/


 1.開発用フォルダを作成（任意）
mkdir react-projects


 2.開発用フォルダにフォーカス
cd react-projects


  3.開発サーバーの作成(create-react-app: Reactアプリ作成ツールでは、すべて自動設定（簡単）)
※Create React App： Reactアプリを簡単に作れる公式ツールです。(複雑な設定を自動で行ってくれます。)
npx create-react-app my-first-app

<コマンド実行結果リスト>
C:\Users\XXXXX\react-projects>npx create-react-app my-first-app



 4.開発サーバー(プロジェクトフォルダ)にフォーカス
cd my-first-app

 5.開発サーバー起動（Reactプログラムの実行）
npm start


<コマンド実行結果リスト>

webpack compiled successfullyの後、自動的にブラウザが起動します。
URL: http://localhost:3000
Reactのロゴがくるくる回る画面が表示されれば成功！


 6.開発サーバーの停止 
停止方法： コマンドプロンプトまたはPowerShellで Ctrl + C を押す

確認メッセージ：
Terminate batch job (Y/N)?
Y を入力して Enter


 7.開発作業 
プロジェクトをVSCodeで開く
方法1: コマンドライン# プロジェクトフォルダで実行
code .

方法2: VSCodeから
1. VSCodeを起動 「ファイル」→「フォルダーを開く」
2. my-first-appフォルダを選択
3. 便利な拡張機能のインストール

■必須の拡張機能：
 1)ES7+ React/Redux/React-Native snippets    Reactのコードを素早く書ける、rce + Tab で関数コンポーネントの雛形
 2)Prettier Code formatter                   コードを自動で整形、保存時に自動実行
 3)Auto Rename Tag                           HTMLタグの開始・終了を同時に変更
 4)Bracket Pair Colorizer 2                  括弧を色分けして見やすくする

 インストール方法：
  1)VSCodeのサイドバーで拡張機能アイコンをクリック
  2)拡張機能名を検索
  3)「インストール」をクリック
